# Hue Fish Themes

Generated Fish shell theme files for prompt integrations.

## Tide

The Tide files live under `tide/`:

- `tide/hue.fish` — entrypoint that sources a mood by `hue_flavour`
- `tide/hue-mua.fish`
- `tide/hue-huong.fish`
- `tide/hue-cung.fish`

These files use `set -g`, not `set -U`, so the theme stays runtime-scoped and
does not write generated colors into `fish_variables`.

Install by sourcing the entrypoint before Tide initializes, for example in
`~/.config/fish/conf.d/00-hue-tide.fish`:

```fish
set -g hue_flavour mua
source ~/.config/fish/themes/hue/tide/hue.fish
```

Use Tide's wizard for layout/icons, but choose true color:

```fish
tide configure --auto --style=Rainbow --prompt_colors='True color'
```

Everything here is generated by the token build. Change
[`packages/tokens/src/adapters/tide.ts`](../tokens/src/adapters/tide.ts) or the
source tokens, then rebuild.
